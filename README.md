This project focuses on analyzing pizza sales data to uncover meaningful business insights using Key Performance Indicators (KPIs) and data visualizations. The goal is to understand sales performance, customer behavior, and product popularity to support data-driven decision-making by using Sql+Excel.

Objective:
1.Track overall business performance using KPIs
2.Identify top 5 best-selling and worst 5 low-performing pizzas
3.Analyze customer order behavior across different stages of the sales funnel
4.Visualize sales trends for quick and clear insights

 Key KPIs Used
Total Revenue,Total Orders,Total Pizzas Sold,Average Order Value (AOV),Average Pizzas per Order
